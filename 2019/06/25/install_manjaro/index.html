Manjaro安装小记&使用体验

2019年6月25日
| 实验室
| 1298 words

5 月的时候，还记得那个夜晚，我激动得睡不着，因为安装了新的系统 Manjaro:joy:。现在已经使用了一个多月，体验很好，比垃圾 Windows 好是肯定的，比 Ubuntu 稳定，软件包更新也更快。感觉可以养老了，不想折腾。

安装 Manjaro

安装的时候都没有截图，所以这一步骤没有图片看的。

当时想在 Ubuntu 下制作一个启动 U 盘来引导，结果发现使用dd命令制作的启动盘没法被 BIOS 识别。之后又想用写 grub menu 直接引导镜像文件，结果也没成功。最后在 Windows 下用某个制作启动盘的软件终于成功了。然后备份文件到另一个硬盘，直接覆盖原先的 Ubuntu 系统。

我选的是 Manjaro + KDE，不过我已经后悔了，因为当时不知道还有 DDE 可以选。KED 太折腾了，DDE 基本不用折腾就很好看了，还有触控板手势可以用。因为我之前就是 Ubuntu + DDE，但是因为图形界面老是会卡死，终于忍无可忍换了 Manjaro。不过现在使用 KDE，也没有发生图形界面卡死的问题。就这样吧。

所以现在笔记本的概况是：

  • Manjaro at 128 GB SSD，固态硬盘不分区。
  • Windows 10 at 500 GB HDD，HDD 分四个区：C、D、E、F。

软件安装

又到了最开心的安装软件时间。。。

选择镜像

sudo pacman-mirrors -i -c China -m rank

添加清华源

编辑 /etc/pacman.conf

[archlinuxcn]
#SigLevel = Optional TrustAll
#SigLevel = Never
Server = http://mirrors.tuna.tsinghua.edu.cn/archlinuxcn/$arch

然后执行sudo pacman -Syu && sudo pacman -S archlinuxcn-keyring 。关于 siglevel 请参考：https://www.archlinux.org/pacman/pacman.conf.5.html#SC

一些国产软件都在这里源里，比如：搜狗输入法、网易云音乐和坚果云。

V2ray

  1. 首先下载并保存安装脚本：curl -L -s https://install.direct/go.sh > go.sh

  2. 直接运行安装：sudo bash go.sh

  3. 如果直接运行下载比较慢，可以先去 GitHub 下载好 v2ray-linux-64.zip

  4. 运行 sudo bash go.sh -l v2ray-linux-64.zip 进行本地安装

  5. 启动V2ray：sudo systemctl start v2ray，开机自启：sudo systemctl enable v2ray

  6. V2ray 配置参考 V2Ray 配置指南

SSR

目前使用 electron-ssr，带有图形界面。添加清华源后可直接安装 sudo pacman -S electron-ssr

输入法

  • fcitx + 搜狗拼音
sudo pacman -S fcitc-im fcitx-configtool fcitx-sogoupniyin

编辑 /etc/environment添加下面的变量，否则会出现无法输入中文的问题。(如果其他程序出现无法输出中文的情况，也可以用此方法，在程序启动脚本中添加如下代码)

export GTK_IM_MODULE=fcitx
export QT_IM_MODULE=fcitx
export XMODIFIERS="@im=fcitx"

MySQL/Mariadb

Install

Install with pacman.

sudo pacman -S mariadb

Initialize data directories.

sudo mysql_install_db --user=mysql --basedir=/usr --datadir=/var/lib/mysql

Run MySQL / MariaDB

Start the service with systemd.

sudo systemctl start mysqld

If you encounter an error like Failed to start mysqld.service: Unit mysqld.service failed to load: No such file or directory. first double check the spelling is correct. If it is then try rebooting and starting the service again.

Enable the service to start on boot.

sudo systemctl enable mysqld

Configure MySQL / MariaDB

Secure the installation.

mysql_secure_installation

Java

Install

yaourt java

Multi Java

archlinux-java status
Available Java environments:
  java-12-jdk (default)
  java-8-openjdk

change default java

archlinux-java set  java-8-openjdk

XMind

yaourt xmind

修复 xmind 打开报错的问题：

编辑/usr/share/xmind/XMind/XMind.ini，删除最后一行的--add-modules=java.se.ee。

在/usr/bin/wps、/usr/bin/wpp、/usr/bin/et文件中都加入下面的代码：
