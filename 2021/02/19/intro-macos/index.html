去年八月份，头脑一热就买了台一直想要的 MacBook Pro。使用几个月后，记录一下从 Manjaro 到 MacOS的差异和使用体验。

基础

  1. 📦安装包管理器 brew：/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"

  2. 系统代理设置：Network->Advanced->Proxies

  3. 输入法：Keyboard -> Input Sources -> Click ➕，竟然自带了小鹤双拼

  4. 修改Terminal size：Terminal -> Preferences -> Window -> Window Size

  5. Remove Search with BaiDu, is it a virus?😂 Change default search engine in Safari

  6. 快捷键：Keyboard -> Shortcus

    1. 把截图并复制到剪切板的快捷键修改成了和Manjora一样的 command+shift+A

    2. ⌘C 复制 ⌘V 粘贴。在终端也可以用哦，不像Linux上要用 Ctrl+Shift+C/V

    3. ⌥option+DelteDelte whole word

    4. Ctrl -> ^control, Alt -> ⌥option

    5. Command + Tab : to switch to windows that are not minimized

    6. 🌟Caps Lock：短按切换中英文输入法；长按锁定字母大写，再次短按取消大写锁定

    7. 其他快捷键改成自己习惯的就行

  7. 屏幕清理：first shut down the computer and unplug the power adapter. Dampen a soft, lint-free cloth with water only, then use it to clean the computer’s screen.

因为 MacOS 也是基于 Unix 的，从 Manjora Linux 迁移过来还是挺顺利的，不论是Terminal还是UI的体验。主要让我不舒服的就是快捷键和键盘的变化，还需要一段时间适应。另外Mac的手势没有我想象的那么惊艳，没有自定义功能，这些手势我在 Manjora 上照样用。唯一一个Mac独有的并且屡试不爽的手势就是three finger drag。

联网软件安装

V2Ray(Move to V2Fly)

https://github.com/v2ray/homebrew-v2ray

  1. brew tap v2ray/v2ray

  2. brew install v2ray-core

  3. vim /usr/local/etc/v2ray/config.json

  4. brew services start v2ray-core

V2Fly

https://github.com/v2fly/v2ray-core

安装：

  1. brew install v2ray

  2. vim /usr/local/etc/v2ray/config.json

  3. brew services start v2ray

编程软件安装

  • Oh My Zsh: sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

  • Java：brew cask install java

  • Golang

    • brew install golang

    • mkdir -p $HOME/go/{bin,src,pkg}

    • export GOPATH=$HOME/go

    • export GOROOT="$(brew --prefix golang)/libexec"

    • export PATH="$PATH:${GOPATH}/bin:${GOROOT}/bin"

  • VS code：官网下载 https://code.visualstudio.com/download

  • IDEA：官网下载

  • Vim 配置：~/.vim/vimrc

    • Vim 和 Zsh 的配置文件都是从Manjora同步过来的，有些不通用的配置需要修改
    • 复制到系统剪切板
      • set mouse =a “鼠标选择进入 VISUAL模式
      • set clipboard =unnamed
      • yy
